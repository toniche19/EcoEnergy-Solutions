# EcoEnergy Solutions

## Descripción

Este proyecto consiste en un análisis exploratorio de datos (EDA) realizado sobre un conjunto de datos de consumo energético de clientes de **EcoEnergy Solutions**.

El análisis se ha desarrollado utilizando **Python** en **Google Colab**, mientras que la visualización de los resultados se ha realizado mediante un panel interactivo en **Google Looker Studio**.

El objetivo principal ha sido identificar patrones de consumo energético, diferencias entre regiones y tipos de contrato, así como analizar las emisiones de CO₂, los costes asociados y el porcentaje de energía renovable utilizado por los clientes.

---

## Objetivos del proyecto

* Analizar el consumo energético de los clientes.
* Comparar el consumo entre regiones.
* Analizar las diferencias entre tipos de contrato.
* Estudiar las emisiones de CO₂.
* Analizar el coste energético.
* Evaluar el porcentaje de energía renovable.
* Diseñar un dashboard interactivo para facilitar la visualización de los resultados.
